HTML5 e CSS3 tecnologias são fundamentais para a criação de páginas modernas web.

HTML5 em é a versão mais recente da linguagem de marcação HTML (HyperText Markup Language). Ele é para usado estruturar o conteúdo de uma página, como, imagens, links e vídeos. HTML5 trouxe muitas melhorias em relação à versão anterior, novos como novos elementosân semticos (exemplo: <heade>,<footer>,<article>,entre outros), suporte nativo a multimídia (como e áudio) e a capacidade de dados de dados localmente não existe.



Já o CSS3 (Cascading Style Sheets) é a usada linguagem para estilizar o conteúdo estruturado com HTML. Com o CSS3, tambémnúmeros de ocupação,números de Ativar, fontes, animações, trans e muito mais. Ele também trouxe novas funcionalidades, como o uso de gradientes, sombras e deções, tornando páginas web dinâmicas e atraentes visualmente.



JavaScript é uma linguagem de programação muito popular, usada principalmente para desenvolver sites e aplicações web. Ele permite que as páginas sejam interativas, como quando clicamos em botões, vemos animações ou atualizamos conteúdo sem precisar recarregar a página.

Com JavaScript, desenvolvedores podem criar desde sites simples até aplicações complexas, como jogos, redes sociais e sistemas de e-commerce. É uma das linguagens essenciais para quem deseja trabalhar com desenvolvimento web.
